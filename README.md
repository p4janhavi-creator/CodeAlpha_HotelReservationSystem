# Hotel Reservation System

## Overview

The Hotel Reservation System is a console-based Java application designed to manage hotel room bookings efficiently. It allows users to search available rooms, create and cancel reservations, view booking details, generate reports, and track hotel statistics.

This project demonstrates the implementation of Object-Oriented Programming (OOP), file handling, collections framework, and Java Date-Time API.

---

## Key Features

- Search available rooms
- Filter rooms by category (Standard, Deluxe, Suite)
- Unique booking ID generation
- Date validation and automatic total cost calculation
- Payment method simulation (Card, GCash, Cash)
- Cancel reservations with status tracking
- Persistent data storage using serialization
- Export booking report to text file
- View hotel statistics (rooms, bookings, revenue)

---

## Technologies Used

- Java
- Java Collections Framework
- Java Time API (LocalDate, DateTimeFormatter)
- File Handling and Serialization
- Stream API

---

## How to Compile and Run

Compile:
javac HotelReservationSystem.java

Run:
java HotelReservationSystem

---

## Data Files Generated

- reservations.dat (stores all reservation records)
- bookings_report.txt (exported booking report)
